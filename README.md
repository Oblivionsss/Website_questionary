# website_questionary
# Проект "Poll Application" на Django
Проект представляет собой простое веб-приложение на платформе Django, предназначенное для автоматизации управления процессами проведения опросов среди студентов. Его цель — обеспечить удобную систему сбора и анализа результатов опросов, как для самих участников, так и для преподавателей.

# Ключевые особенности проекта:
Автоматизированная административная панель: преподаватели могут создавать новые опросы, управлять ими и анализировать полученные результаты.
Удобный интерфейс для пользователей: участники могут удобно видеть доступные опросы и отправлять свои ответы.
Система аналитики: автоматический сбор и обработка данных позволяет быстро получать отчёты по результатам опросов.
Архитектура проекта:
Проект построен на стандартной структуре Django
Polls: модуль, отвечающий за логику опроса, обработку данных и отображение результатов.
Templates: шаблоны для рендеринга интерфейса.
Models: определение схем данных и бизнес-логики.
Views: обработчики запросов, взаимодействующие с моделями и передающие данные в шаблоны.
# Инструкция по установке и запуску:
Склонируйте репозиторий: git clone git@github.com:<your_username>/my-poll-app.git
# Установите виртуальное окружение и требуемые пакеты: 
python -m venv env;
source env/bin/activate;
pip install -r requirements.txt
# Примените миграции базы данных:
python manage.py makemigrations
python manage.py migrate
# Запустите сервер разработки:
python manage.py runserver
После этого приложение станет доступным по адресу http://127.0.0.1:8000.
